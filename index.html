Introduction

JavaScript is a versatile programming language primarily for web development.

It allows developers to create interactive and dynamic web pages.

Introduced in 1995, it has become a core technology of the World Wide Web.

console.log("Hello, World!");
Variables

Variables are containers for storing data values in JavaScript.

You can declare variables using keywords such as var, let, or const.

Variables declared with let or const are block-scoped.

let x = 5;
  • var: Function-scoped and can be re-declared.
  • let: Block-scoped and cannot be re-declared in the same scope.
  • const: Block-scoped and cannot be re-assigned or re-declared.
Functions

Functions are blocks of code designed to perform specific tasks.

They are executed when invoked (called).

Functions can take parameters and return values.

function greet(name) { return `Hello, ${name}`; }
Loops

Loops allow repetitive execution of a block of code.

JavaScript supports for, while, and do...while loops.

  • The for loop runs a block of code a specific number of times.
  • The while loop runs as long as a condition is true.
  • The do...while loop runs at least once before checking the condition.
for (let i = 0; i < 5; i++) { console.log(i); }
Arrays

Arrays are special variables that can hold multiple values in JavaScript.

Each value in an array is stored as an element and accessed using an index.

Arrays provide methods to manipulate and iterate through their elements.

let fruits = ["apple", "banana", "cherry"];
